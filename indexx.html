MENGENAL CYBER SECURITY

APA ITU CYBER SECURITY?

cyber security (keamanan siber) adalah bidang yang berfokus pada melindungi komputer, jaringan, sistem, dan data dari serangan digital seperti hacker, malware, phishing, dan pencurian data, dengan tujuan menjaga keamanan informasi di dunia digital; hal ini dilakukan melalui berbagai cara seperti menggunakan password yang kuat, mengaktifkan verifikasi dua langkah (2FA), memakai antivirus, tidak sembarangan mengklik link mencurigakan, serta rutin memperbarui sistem, dan di dalamnya terdapat beberapa bidang seperti ethical hacking, network security, digital forensics, dan application security yang semuanya sangat penting karena hampir semua aktivitas sekarang dilakukan secara online sehingga tanpa perlindungan yang baik, data pribadi, akun, dan transaksi bisa dengan mudah disalahgunakan.

FUNGSI CYBER SECURITY

Fungsi cyber security (keamanan siber) adalah untuk melindungi sistem komputer, jaringan, dan data dari berbagai ancaman digital seperti peretasan, virus, dan pencurian informasi, sehingga data pribadi maupun organisasi tetap aman, tidak disalahgunakan, dan tetap bisa diakses oleh pihak yang berhak; selain itu, cyber security juga berfungsi menjaga privasi pengguna, mencegah kerugian finansial, memastikan sistem berjalan dengan lancar tanpa gangguan, serta mendeteksi dan menangani serangan jika terjadi sehingga keamanan digital tetap terjaga.

MANFAAT CYBER SECURITY

Manfaat cyber security (keamanan siber) adalah menjaga data dan informasi agar tetap aman dari pencurian atau penyalahgunaan, melindungi privasi pengguna saat beraktivitas di internet, mencegah kerugian finansial akibat penipuan atau peretasan, menjaga kinerja sistem agar tetap stabil dan tidak terganggu oleh virus atau serangan, serta meningkatkan rasa aman dan kepercayaan saat menggunakan teknologi digital baik untuk belajar, bekerja, maupun bertransaksi online.

CONTOH CODINGAN

import sqlite3

conn = sqlite3.connect("database.db")
cursor = conn.cursor()

username = input("Username: ")
password = input("Password: ")

cursor.execute("SELECT * FROM users WHERE username=? AND password=?", (username, password))

result = cursor.fetchone()

if result:
    print("Login berhasil")
else:
    print("Login gagal")
